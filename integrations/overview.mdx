TapKit connects to the AI tools you already use. No custom code required — install a plugin or add an MCP server URL, and your AI agent can control iPhones.

## What's the same everywhere

Every integration gives the same core capabilities: tap, swipe, type, launch apps, take screenshots, and stream the phone screen. The platform changes how you talk to TapKit, not what it can do.

<Info>
  **All integrations require the TapKit Mac app running with at least one phone connected.** If you haven't set that up yet, start with [Mac App Setup](/setup/mac-app) and [Phone Setup](/setup/phone).
</Info>
