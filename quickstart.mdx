This guide walks you through the fastest path from zero to controlling an iPhone with TapKit. Each step links to a detailed setup page if you need more help.

## Step 1: Create an account

Sign up at [tapkit.ai](https://tapkit.ai) and grab your API key from the dashboard.

<Card title="Open Dashboard" icon="arrow-up-right-from-square" href="https://tapkit.ai/dashboard">
  Sign up or sign in to get your API key.
</Card>

## Step 2: Download the Mac app

The TapKit Mac app is the host that connects physical iPhones to your account. Download it, install it, and sign in.

1. Download the TapKit Mac app from your [dashboard](https://tapkit.ai/dashboard)
2. Open the DMG and drag TapKit to Applications
3. Launch TapKit and sign in with your account

<Note>
  Need help? See the full [Mac App Setup guide](/setup/mac-app) for detailed instructions including permissions and troubleshooting.
</Note>

## Step 3: Connect an iPhone

Plug your iPhone into your Mac via USB and follow the setup flow:

1. **Plug in your iPhone** via USB — the Mac app detects it automatically
2. **Trust the connection** when prompted on both devices
3. **Scan the QR code** shown in the Mac app with your iPhone camera — this runs the setup Shortcut that configures your phone's settings
4. **Enable Switch Control** when directed at the end of the setup flow
5. **Add the TapKit Shortcut** by scanning the second QR code and following the prompts

<Note>
  For detailed instructions with screenshots, see the full [Phone Setup guide](/setup/phone).
</Note>

## Step 4: Run your first task

Your Mac app is open and your phone is connected. Use the built-in agent — type a natural language command and watch the phone do it:

> "Open Safari and search for the weather in San Francisco"

The agent takes a screenshot, reasons about what it sees, performs an action, and repeats — all visible in real-time on the phone screen.

## Step 5: Integrate with your AI tools

Now that you've seen TapKit in action, connect it to the tools you already use:

- **Claude.ai / Co-work** — add TapKit as an MCP connector in Claude settings, zero code required
- **Claude Code** — install the TapKit plugin for terminal-based iPhone control
- **Codex** — connect via the Codex CLI or app (coming soon)
- **Any MCP client** — point your client at TapKit's MCP server URL
