---
title: Escape
description: Press escape / go back
api: POST /v1/phones/{phone_id}/escape
---

Press escape to navigate back or dismiss the current modal/dialog.

## Request

```bash
curl -X POST https://api.tapkit.ai/v1/phones/{phone_id}/escape \
  -H "X-API-Key: joot_your_api_key"
```

No request body required.

```json
{
  "id": "job_abc123",
  "status": "completed",
  "result": {},
  "created_at": "2024-01-15T10:30:00Z",
  "completed_at": "2024-01-15T10:30:01Z"
}
```
