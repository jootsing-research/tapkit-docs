App skills map out an iOS app's entire UI — tabs, buttons, gestures, navigation patterns, and common gotchas. The agent reads these to understand *where things are* when it sees a screenshot.

## What's inside an app skill

Each skill contains:

- **App structure** — tabs, UI hierarchy, screen layouts
- **Interaction patterns** — how to navigate, which gestures to use
- **Common workflows** — step-by-step flows for frequent tasks
- **Gotchas** — quirks, loading states, edge cases the agent should know about
