---
title: Installation
description: Install the TapKit CLI via curl or Homebrew
---

TapKit CLI is a native macOS binary. No runtime dependencies — no Python, no Node.js, no Docker.

<Tabs>
  <Tab title="Native Installer (recommended)">
    ```bash
    curl -fsSL https://tapkit.ai/install.sh | bash
    ```

    This downloads the latest `tapkit` binary to `/usr/local/bin` and sets up auto-updates.
  </Tab>
  <Tab title="Homebrew">
    ```bash
    brew install tapkit
    ```

    Homebrew installs don't auto-update. Run `brew upgrade tapkit` for new versions.
  </Tab>
</Tabs>

Verify the installation:

```bash
tapkit --version
# tapkit 1.0.0 (arm64-apple-macos14)
```

<Info>
  **Prerequisites** — TapKit requires a **Mac** running the [TapKit companion app](https://tapkit.ai/download) with an **iPhone connected** via USB or Wi-Fi. The CLI talks to the TapKit API, which routes commands through the companion app to your phone.
</Info>
