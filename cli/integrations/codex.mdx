---

The TapKit plugin for Codex gives the agent iPhone control in your terminal. It installs an MCP server connection and skills for app knowledge.

## Setup

Add the TapKit marketplace from your terminal:

```bash
codex plugin marketplace add jootsing-research/tapkit-plugins-codex
```

Open the Codex plugin directory:

```bash
codex
/plugins
```

Switch to the **Tap Kit Plugins** marketplace, open **Tap Kit**, and select **Install plugin**.

## Upgrade

Refresh the TapKit marketplace when a new plugin version is available:

```bash
codex plugin marketplace upgrade tapkit-plugins
```

`tapkit-plugins` is the Codex marketplace name from the marketplace metadata.

## Usage

Codex calls TapKit through the bundled MCP tools. The skills teach it the screenshot → look → act → verify workflow:

```
you: Take a screenshot of my phone and tell me what app is open

codex: Running TapKit screenshot tool
        I can see Twitter is open on the Home feed. The top tweet is from
        @elonmusk about... Would you like me to interact with anything?
```

The core skill teaches Codex how to use the coordinate system, chain actions, and verify results — so it knows the workflow without you having to explain it.
